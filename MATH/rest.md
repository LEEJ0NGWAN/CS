# 나머지 정리

### 덧셈

`(a + b) % p = (a%p + b%p) % p`

e.g., (7 + 6) % 5 = (7%5 + 6%5) % 5 = 3

### 뺄셈

`(a - b) % p = (a%p - b%p) % p`

### 곱셈

`(a * b) % p = (a%p * b%p) % p`

## 컴비네이션에 대한 나머지 정리

nCr % p

= (n! * 1/(n-r)! * 1/r!) % p

= (n! % p * 1/(n-r)! % p * 1/r! % p) % p

= (n! % p * ((n-r)!)^(p-2) % p * (r!)^(p-2) % p) % p

n! % p

= (n * (n-1)!) % p

= (n%p * (n-1)! % p) % p

= (n%p * (n-1)%p * ... * 1%p) % p

n! ^ x % p

= (n! * n! * ... * n!) % p

= (n! % p * ... * n! % p) % p
